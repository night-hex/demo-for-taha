# 🌸 demo-for-taha

A small interactive portfolio demo for Taha Qureshi with a soft pink theme.

- **Open the site:** double-click or serve `index.html` and view in your browser.
- **Files:** `index.html` contains the full single-file portfolio.

> Quick preview

1. Start a simple static server (Python):

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
